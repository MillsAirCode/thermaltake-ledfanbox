# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-28

### Added
- Initial release of Thermaltake LEDFanBox Controller
- Fan speed control (0-100%)
- RGB color control with hex values and color names
- Multiple lighting modes (Off, Static, Breathing, Rainbow)
- Color cycling mode
- Comprehensive documentation
- Apache License 2.0
- Setup.py for package installation
- Systemd service configuration
- Git repository templates (ISSUE_TEMPLATE, PR_TEMPLATE)
- Contributing guidelines

### Fixed
- Fixed Python bytes conversion error in fan speed command
- Improved error handling and user-friendly messages

### Supported Platforms
- Linux (Ubuntu, Debian, etc.)
- macOS
- Windows

### Supported Features
- Automatic device detection
- Real-time color control
- Command-line interface
- HSV color conversion
- Cross-platform HIDAPI support
